About

Hi, I'm Chris.

I graduated from CSU Long Beach in 2022 with a BA in Theater Design and Technology.
I like to draw, read, surf, hike, run (kinda), bike around, design, and build stuff (physically and digitally).
I also like solving problems and learning new things to help me solve more interesting problems.

I currently know: html+css, bash, php + js fundamentals, and how to set up a venv
I'm currently learning: ML foundations with fast.ai
I'd like to learn: c, js

My favorite part about digital stuff is that anything which can be imagined and thought out can be built. Working through the design process and seeing ideas move from a mental visual to a finished product is a reward cycle for me, and eventually, I'd like to be able to build the most neat and interesting stuff.

I also like reading neat and interesting stuff. If you have any inclination to suggest a book, please feel free.

If you've made it this far, thank you, and I hope you have a good day.
